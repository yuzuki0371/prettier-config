# @yuzuki0371/prettier-config

Shared [Prettier](https://prettier.io/) configuration.

## Install

```sh
npm install --save-dev @yuzuki0371/prettier-config
```

## Usage

`package.json` に以下を追加:

```json
{
  "prettier": "@yuzuki0371/prettier-config"
}
```

## Rules

| Option | Value |
| --- | --- |
| `printWidth` | `120` |
| `quoteProps` | `"consistent"` |
| `trailingComma` | `"es5"` |
| `arrowParens` | `"avoid"` |
